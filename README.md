# integration-practice

Проект демонстрирует разрабоку требований, интеграцию модулей, отладку, тестирование и инспекцию кода.

## Структура проекта

- `src/integration_practice` — модули приложения (`DataProvider`, `Processor`, `Orchestrator`).
- `tests` — автоматизированные тесты для модулей.
- `docs` — требования, тестовый план и результаты инспекции.
- `requirements.txt` — зависимости для запуска тестов и проверки качества.

## Быстрый старт

1. Установите зависимости:

```bash
pip install -r requirements.txt
```

2. Запустите приложение:

```bash
python main.py
```

3. Запустите тесты:

```bash
pytest
```

4. Проверьте код на соответствие стандартам:

```bash
flake8 src tests
```

## Документация

- `docs/module_requirements.md` — требования к модулям и взаимодействию компонентов.
- `docs/test_plan.md` — тестовые наборы и сценарии.
- `docs/code_inspection.md` — результат инспектирования и правила кодирования.
